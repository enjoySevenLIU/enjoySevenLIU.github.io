位运算符

用法和C#基本一致：CS1L13——位运算符

  • 位与：&​ (同为1则1，有0则0)
  • 位或：|​ (有1则1)
  • 异或：^​ (相同为0，不同为1)
  • 取反：~​ (1变0, 0变1)
  • 左移：<<​
  • 右移：>>​ (最高位保留正负，即最高位为1就补1，最高位为0就补0)
  • 无符号右移：>>>​ (最高位始终补0)
