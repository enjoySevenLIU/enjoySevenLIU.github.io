P2L9——【程序准备工作】Github注册登录 Gitbub Desktop的下载安装

更好的体验github的一种方式

由于某些原因，github的直接使用体验不甚理想，想要稳定流畅使用需要借助加速工具，
这里可以使用羽翼城大佬制作的steamcommunity 302，它是免费的，在这里下载：steamcommunity 302 – Dogfight360
该工具虽然主要用于加速Steam社区的浏览，但该工具内置了很多网页的加速选项，其中就包括github，且笔者实测效果显著

在下载安装并打开该工具后，点击设置

​​

在本地反代选择处选择Github访问，保存设置重启软件即可

​​

加速Github的访问有多种方式，本方式仅供参考

Github注册登录

**1. 访问github网站：**​GitHub

2. 点击右上角登录

​​

3.若没有账号，则注册

​​

4.注册流程

邮箱输入

​​

密码输入

​​

用户名输入

​​

邮箱是否接收产品更新和公告，根据需求选择

​​

防机器人验证

​​

回答问题即可

​​

登录注册用的邮箱，收取验证码后输入

​​

5.登录

​​

6.首次登录问题填写

问题一：有多少团队成员和你一起工作，根据实际情况选择即可

问题二：你是否是学生或老师，根据实际情况填写即可

​​

问题三：你准备使用哪些功能，根据实际情况选择即可（看不懂就翻译过来看）

​​

Collaborative coding：协作编码（建议选择）

Automation and CI/CD：自动化 CI/CD

Security：安全性

Client Apps：客户端应用程序（建议选择）

Project Management：项目管理（建议选择）

Team Administration：团队管理（建议选择）

Community：社区使用（建议选择）

7.登录设置成功，进入个人页面

​​

8.让Github页面变为中文显示

方式一：使用浏览器自带翻译功能（不赘述，建议没有安装过浏览器拓展插件的同学使用方式一即可）

方式二：使用浏览器插件

点击此处搜索Github Chinese

​​

​​

找到一个叫做maboloshi/github-chinese的汉化库，点击它

​​

来到对应页面后，往下滑

​​

按照下方的流程进行安装（若没有安装过浏览器拓展插件的同学，可能操作起来比较麻烦，可以暂时不弄，使用方式一即可）

​​

下载安装Github Desktop

1.在上节课登录成功后的个人界面中

点击安装Github Desktop

​​

点击下载按钮，选择存储路径后下载（也可以在资料区获取安装包）

​​

下载成功后，双击打开

​​

弹出界面后，点击登录

​​

由于网页上我们已经登录过，直接点击授权桌面应用即可

​​

此时会弹出该窗口，点击打开即可

​​

此时窗口会更新，这里可以选择是不是要使用你的github账户名和email地址，选择使用即可

也可以选择Configure manually（手动配置）

我们选择第一个后，点击下方的Finish按钮

​​

之后便会弹出仓库管理窗口了

​​

桌面也会出现图标

​​

安装完成

创建新的代码仓库
