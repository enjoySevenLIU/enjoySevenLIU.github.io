U3L16-1-b——2D Animation图集骨骼编辑

注意事项

  1. 设置Sprite为图集模式
  2. 对图集图片进行切片

图集骨骼编辑

  1. 选择好图集中单个要修改的图
  2. 按照单张图片编辑骨骼的方法，来为图集内每一个图片增加骨骼和蒙皮
  3. 最后将这些图片放在场景内一个根物体下，拼在一起，注意层级和父子关系

图集骨骼动画使用

拼好的骨骼使用 和 骨骼动画使用 一致
