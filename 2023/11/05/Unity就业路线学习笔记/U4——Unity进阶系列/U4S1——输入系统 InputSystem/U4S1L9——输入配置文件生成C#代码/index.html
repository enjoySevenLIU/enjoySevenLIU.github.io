U4S1L9——输入配置文件生成C#代码

之前我们通过配置文件，可以做到一个配置文件管理多个 InputAction​ 类，那我们该如何在C#脚本里使用这些类呢？
我们需要通过配置文件生成自动一个C#脚本，使我们可以让配置文件配置的输入方案使用C#代码监听

根据配置文件生成C#代码

  1. 选择 InputActions 文件
  2. 在 Inspector 窗口设置生成路径，类名，命名空间
  3. 应用后生成代码
  4. 注意生成的脚本名，不要将已有的C#脚本覆盖了！
    之后我们也要通过这个脚本中的类来监听输入
