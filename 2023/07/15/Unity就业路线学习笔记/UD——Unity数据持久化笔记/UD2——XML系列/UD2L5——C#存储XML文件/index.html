决定存储在哪个文件夹下

注意！存储xml文件 在Unity中一定是使用各平台可读可写可找到的路径

  1. ​Resources​ 可读 不可写 打包后找不到 x
  2. ​Application.streamingAssentsPath​ 可读 PC端可写 找得到 x
  3. ​Application.dataPath​ 打包后找不到 x
  4. ​Application.persistentDataPath​ 可读可写找得到 √

存储xml文件

  • 关键类 XmlDocument​ 用于创建节点 存储文件
  • 关键类 XmlDeclaration​ 用于添加版本信息
  • 关键类 XmlElement​ 节点类
