U3L3——Sprite Editor

Sprite Editor

顾名思义，Sprite Editor 就是 精灵图片编辑器
它主要用于编辑2D游戏开发中使用的Sprite精灵图片
它可以用于编辑 图集中提取元素，设置精灵边框，设置九宫格，设置轴心（中心）点等等功能

安装Sprite Editor

新版本Unity，如果在3D工程内使用Sprite Editor
需要安装 2D Sprite包才能使用Sprite Editor

Sprite Editor的编辑内容

  • Single图片编辑
  • Multiple图集元素分割
  • Polygon多边形编辑
