CS1L15——if语句

if

作用

让顺序执行的代码 产生分支
if语句是一个可以让程序产生逻辑变化的语句

if

作用：满足条件时多执行一些代码

注意：if语句的语法部分不需要写 “;​”，if​可以嵌套使用

if…else…

作用：产生两条分支 满足条件做if内代码块，不满足则做else内代码块

注意：

  1. ​if...else​ 语句的语法部分不需要写分号
  2. ​if...else​ 语句可以嵌套

if… else if… else if… else…

作用：产生n条分支 多条道路选择 最先满足其中的一个条件 就做什么
语法：

注意：

  1. 和前面两个是一样的 不需要写分号
  2. 是可以嵌套的
  3. ​else​ 语句块可以省略
  4. 条件判断 从上到下执行 满足了第一个之后 之后的都不会执行了
